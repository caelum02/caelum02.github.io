컨벡스 최적화란?

Haneul Choi on Dec 252021-12-25T23:35:21+09:00
Updated Dec 302021-12-30T07:20:00+09:00 3 min read

송현오 교수님의 Machine Learning 강의 자료를 참고하여 작성하였습니다.

1. Optimization Problem

  • Optimization Problem은 주어진 조건들을 만족할 때, $x$에 대한 어떤 함수 $f_0$의 최솟값을 구하는 Problem의 Class이다.
  • 수학적으로 보통 다음과 같이 표현한다.
    \[ \min_{\boldsymbol{x}} f_0(\boldsymbol{x}) \] \[ \text{subject to } f_i(x) \leq b_i, (i=1,\dots,m) \]
  • optimization variable $\boldsymbol{x} \in \mathrm{R}^n$, objective function $f_0: \mathrm{R^n} \rightarrow \mathrm{R}$, constraint function $f_i: \mathrm{R}^n \rightarrow \mathrm{R}$

2 Convex Optimization Problem

2.1 정의

  • 일반적인 Optimization Problem과 달리, $f_0,\dots,f_m$가 convex한 problem class
  • 조건 하에서 $f_0$의 local minima는 반드시 global minima가 됨이 보장됨.
  • local minimia를 찾는 것이 global minima를 찾는 것보다 훨씬 쉽기 때문에, Convex Optimization Problem은 다른 Optimization Problem에 비해 global minima를 쉽게 찾을 수 있음

2.1.1 cf

  • 제약조건 하에서 $f_0$의 정의역은 $\Omega = \{\boldsymbol{x}\in \mathrm{R}^n | f_i(\boldsymbol{x}) \leq b_i, (i=1,\dots,m) \}$이다.
  • 각 함수의 epigraph는 convex, convex set의 교집합은 convex하므로 $\Omega$는 convex, $f_0$는 convex function이 됨.

2.2 집합의 Convexity

  • 집합 $S$ 내의 임의의 두 점을 있는 선분이 모두 다시 집합에 $S$ 포함될 때, 집합 $S$는 Convex Set이다.
  • i.e.) $\forall \boldsymbol{x, y} \in S, \theta \in [0, 1], \theta \boldsymbol{x} + (1-\theta) \boldsymbol{y} \in S$
  • 직관적으로, 오목하게 들어간 부분이 없는 집합

2.2.1 예시

  • empty set, $\mathrm{R}^n$
  • Convex hull
  • Convex corn (원점에서 Convex hull에 빛을 쏘면 생기는 그림자와 같은 영역)

2.3 함수의 Convexity

  • Convex set $\Omega$를 정의역으로 갖는 함수 $f: \Omega \rightarrow \mathrm{R}$가 다음을 만족하면, convex function이다.
  • $f(\theta \boldsymbol{x} + (1-\theta)\boldsymbol{y}) \leq \theta f(\boldsymbol{x}) + (1-\theta)f(\boldsymbol{y})$
  • $f$의 정의역이 convex set이 아니면, 위 조건의 좌변이 애초에 불가능함

2.3.1 예시

  • convex function들의 pointwise maximum, Affine 변환, non-negative weighted sum 등은 함수의 Convexity를 보존한다.

2.4 Local minima는 Global Minima

  • $x$가 $\Omega$에서 정의된 convex function $f$의 local minima라 하자.
  • local minmality로부터, $\epsilon > 0$이 존재해 모든 $y \in \Omega$에 대해 다음 성립
    \[ \begin{aligned} f(x) &\leq f((1-\epsilon)x+\epsilon y) && (\text{local minima})\\
    &\leq (1-\epsilon)f(x)+\epsilon f(y) \\
    &\leq f(x) + \epsilon (f(y)-f(x)) \\\\\
    \therefore f(x) &\leq f(y) \end{aligned} \]
